Adressverwaltung

Die Adressverwaltung kann über den Reiter "Adressverwaltung" in der Navigationsleiste erreicht werden.
Sie dient dazu die Personen und Organisationen des Systems zu verwalten.

Warenverwaltung

Die Warenverwaltung kann mithilfe des Reiters "Warenverwaltung" geöffnet werden.
Hier können Wareneingänge, Warenausgänge und Lieferlisten erstellt, bearbeitet, gelöscht und exportiert werden.

Reporting

Das Reporting kann mithilfe des Reiters "Warenverwaltung" erreicht werden.
Es dient dazu Reports über eingegangene Waren, ausgegangene Waren und Personen zu generieren.

Personenverwaltung

Um zur Personenverwaltung zu gelangen muss in der Adressverwaltung der Reiter "Personen" ausgewählt werden.

In der Tabelle werden alle im System befindlichen Personen angezeigt.


Personen filtern:

Mit dem Suchfilter können die Personen nach beliebigen Stichwörtern durchsucht werden. Der Filter wird automatisch bei Eingabe aktiv und listet nur mehr die Personen auf, die den gewünschten Begriff in einer der Spalten aufweisen. Weiters können die Personen nach ihren Typen gefiltert werden indem die Checkboxen an- bzw. abgewählt werden.


Eine Person neu anlegen / bearbeiten:

Mithilfe der Buttons "Neu anlegen" und "Bearbeiten" kann man eine neue Person anlegen bzw. eine bestehende Person editieren. Es erscheint ein Dialog in dem die notwendigen Daten eingegeben werden können. Der Nachname ist ein Pflichtfeld und muss somit ausgefüllt werden. Alle anderen Felder können bei Bedarf leer gelassen werden.
Mithilfe der Checkbox "Systembenutzer" kann ausgewählt werden, ob die zu erstellende / ändernde Person Zugriff auf das System erhalten soll. Es muss hierzu eine der angegebenen Email-Adressen als Login ausgewählt werden. Weiters können die zutreffenden Rechte vergeben werden.


Eine Person löschen:

Mithilfe des Buttons "Löschen" kann eine Person aus dem System gelöscht werden. Der jeweilige Eintrag wird in der Datenbank als inaktiv vermerkt und erscheint auch nicht mehr im System. Sollte die als inaktiv vermerkte Person jedoch wieder aktiviert werden sollen, so muss dies direkt in der Datenbank geschehen.


Personendetails anzeigen:

Der Button "Details" dient dazu die Personendetails einer Person anzuzeigen. Hier werden alle vorhandenen Informationen zur gewünschten Person dargestellt.


Alle Personen exportieren:

Mit dem Button "Exportieren" kann eine vollständige Liste aller im System erfassten Personen als .csv – Datei generiert werden. Diese Liste enthält sowohl aktive als auch inaktive Personen.

Organisationsverwaltung

In der Tabelle werden alle im System befindlichen Organisationen angezeigt.


Organisationen filtern:

Mit dem Suchfilter können die Organisationen nach beliebigen Stichwörtern durchsucht werden. Der Filter wird automatisch bei Eingabe aktiv und listet nur mehr die Organisationen auf, die den gewünschten Begriff in einer der Spalten aufweisen. Weiters können die Organisationen nach ihren Typen gefiltert werden indem die Checkboxen an- bzw. abgewählt werden.


Eine Organisation neu anlegen / bearbeiten:

Mithilfe der Buttons "Neu anlegen" und "Bearbeiten" kann man eine neue Organisation anlegen bzw. eine bestehende Organisation editieren. Es erscheint ein Dialog in dem die notwendigen Daten eingegeben werden können. Der Organisationsname ist ein Pflichtfeld und muss somit ausgefüllt werden. Alle anderen Felder können bei Bedarf leer gelassen werden.

Es werden jedoch nur Personen und Kategorien angezeigt, die bereits im System angelegt sind. Soll eine Kontaktperson zugeordnet werden, die es nicht im System gibt, muss dieses zuerst in der Personenverwaltung angelegt werden. Dies gilt ebenso für Kategorien, die noch nicht im System erfasst wurden. Weiters kann mithilfe der Checkboxen festgelegt werden, welchen Typ eine Organisation hat.

Organisationsdetails anzeigen:

Der Button "Details" dient dazu die Organisationsdetails einer Organisation anzuzeigen. Hier werden alle vorhandenen Informationen zur gewünschten Organisation dargestellt.


Alle Organisationen exportieren:

Kategorieverwaltung

Um zur Kategorieverwaltung zu gelangen muss in der Adressverwaltung der Reiter "Kategorien" ausgewählt werden.

In der Tabelle werden alle im System angelegten Kategorien angezeigt.


Kategorien durchsuchen:

Mit dem Suchfilter können die Kategorien nach beliebigen Stichwörtern durchsucht werden. Der Filter wird automatisch bei Eingabe aktiv und listet nur mehr die Kategorien auf, die den gewünschten Begriff aufweisen.


Eine Kategorie neu anlegen / bearbeiten:

Mithilfe der Buttons "Neu anlegen" und "Bearbeiten" kann man eine neue Kategorie anlegen bzw. eine bestehende Kategorie editieren. Es erscheint ein Dialog in dem ein Kategoriename und eine Kategoriebeschreibung eingegeben werden können. Der Kategoriename ist ein Pflichtfeld und muss somit ausgefüllt werden. Die Beschreibung kann bei Bedarf leer gelassen werden. Sie dient dazu eine ausführlichere Beschreibung hinzuzufügen.


Eine Kategorie löschen:

Mithilfe des Buttons "Löschen" kann eine Kategorie aus dem System gelöscht werden. Es erscheint ein Dialog, mit dem die Aktion bestätigt werden kann. Weiters wird in dem Dialog angezeigt, welche Organisationen diese Kategorie zugewiesen haben. Denn das Löschen einer Kategorie führt dazu, dass ebenfalls die Zuordnung dieser Kategorie für alle zutreffenden Organisationen entfernt wird.

Wareneingang

Um zur Übersicht über die Wareneingänge zu gelangen, muss in der Warenverwaltung der Reiter "Wareneingang" ausgewählt werden.

In der Tabelle werden alle nicht-archivierten im System befindlichen Wareneingänge angezeigt.
Eine graue Hinterlegung eines Wareneingangs bedeutet, dass dieser Artikel enthält, die bereits verbucht worden sind.
Daher ist das Bearbeiten nur mehr geringfügig möglich und das Löschen dieser nicht verfügbar.


Wareneingänge durchsuchen:

Mit dem Suchfilter können die Wareneingänge nach beliebigen Stichwörtern durchsucht werden. Der Filter wird automatisch bei Eingabe aktiv und listet nur mehr die Wareneingänge auf, die den gewünschten Begriff aufweisen.
Somit kann beispielsweise nache gewissen Lieferanten, Artikeln und Lieferdaten gesucht werden.


Archivierte Wareneingänge anzeigen:

Archivierte Wareneingänge können durch Anwählen der Checkbox "Archivierte Einträge" angezeigt werden. In der letzten Tabellenspalte wird der Archivierungsstatus dargestellt.


Einen neuen Wareneingang anlegen / bearbeiten:

Mithilfe der Buttons "Neuer Wareneingang" und "Bearbeiten" gelangt man zu einer neuen Maske, die das Erfassen bzw. Bearbeiten des Wareneingangs ermöglicht.


Einen Wareneingang löschen:

Mithilfe des Buttons "Löschen" kann der ausgewählte Wareneingang gelöscht werden. Es erscheint ein Dialog, mit dem die Aktion bestätigt werden kann. Der Wareneingang kann nur gelöscht werden, wenn dieser noch nicht verbucht worden ist.


Einen Wareneingang archivieren:

Mithilfe des Buttons "Archivieren" bzw. "De-Archivieren" kann der ausgewählte Wareneingang archiviert werden bzw. die Archivierung rückgängig gemacht werden.


Details anzeigen:

Mithilfe des Buttons "Details" können alle verfügbaren Details zu dem ausgewählten Wareneingang angezeigt werden.


Verteilungsreport exportieren:

Durch Klicken auf den Button "Verteilungsreport exportieren" kann ein detaillierter Report über die Verteilung aller Artikel des gewählten Wareneingangs generiert werden.
In diesem Report werden alle Artikel des Wareneingangs aufgelistet und alle erhaltenden Organisationen mit der jeweiligen Anzahl an Verpackungseinheiten dargestellt.

Neuer Wareneingang

Um einen neuen Wareneingang zu erstellen, muss im Reiter "Wareneingang" auf den Button "Neuer Wareneingang" geklickt werden.

Der Vorgang kann abgebrochen werden indem auf das X des Reiters geklickt wird, oder ein anderer Reiter ausgewählt wird.


Lieferant:

Hier kann der gewünschte Lieferant für den Wareneingang gewählt werden. In dem Auswahlfenster kann nach Name, Stadt und Kategorien gesucht werden.
Der gewünschte Lieferant wird ausgewählt indem die entsprechende Checkbox rechts neben dem Eintrag gewählt wird. Anschließend wird die Auswahl mit Speichern bestätigt.


Datum:

Das Datum der Lieferung kann mithilfe der Auswahlbox gewählt werden. Optinal kann das Datum auch in der Form "dd.mm.yyyy" eingegeben werden.


Bemerkung:

In diesem Feld kann dem Wareneingang eine beliebige Bemerkung hinzugefügt werden.


Eine neue Position hinzufügen:

Dem Wareneingang kann durch Klick auf den Button "Neue Position" ein neuer Eintrag hinzugefügt werden. Über das erscheinende Fenster können alle notwendigen Informationen zur Position eingetragen werden. Alle Felder bis auf den Einzelpreis sind Pflichtfelder und müssen ausgefüllt werden.
Der neue Eintrag wird mittels "Speichern" bestätigt und dem Wareneingang hinzugefügt.


Eine bestehende Position bearbeiten:

Mithilfe des Buttons "Bearbeiten" kann die ausgewählte Position bearbeitet werden. Es erscheint ein Fenster in dem die Änderungen getätigt werden können.


Eine Position löschen:

Mithilfe des Buttons "Löschen" kann die ausgewählte Position entfernt werden.


Positionen sortieren:

Mithilfe der Pfeilbuttons kann der gewählte Eintrag innerhalb des Wareneingangs sortiert werden.


Wareneingang speichern:

Mithilfe des Buttons "Alles zum Depot hinzufügen" kann der Wareneingang im System gespeichert werden. Alle Eingaben werden erst hierdurch gültig.

Wareneingang bearbeiten

Um einen bestehenden Wareneingang zu bearbeiten, muss im Reiter "Wareneingang" ein Wareneingang ausgewählt werden und auf den Button "Bearbeiten" geklickt werden.

Der Vorgang kann abgebrochen werden indem auf das X des Reiters geklickt wird, oder ein anderer Reiter ausgewählt wird.


Lieferant:

Hier kann der gewünschte Lieferant für den Wareneingang gewählt werden. In dem Auswahlfenster kann nach Name, Stadt und Kategorien gesucht werden.
Der gewünschte Lieferant wird ausgewählt indem die entsprechende Checkbox rechts neben dem Eintrag gewählt wird. Anschließend wird die Auswahl mit Speichern bestätigt.
Achtung: Der Lieferant kann nur geändert werden, wenn der Wareneingang nocht nicht verbucht wurde.


Datum:

Das Datum der Lieferung kann mithilfe der Auswahlbox gewählt werden. Optinal kann das Datum auch in der Form "dd.mm.yyyy" eingegeben werden.


Bemerkung:

In diesem Feld kann dem Wareneingang eine beliebige Bemerkung hinzugefügt werden.


Eine neue Position hinzufügen:

Dem Wareneingang kann durch Klick auf den Button "Neue Position" ein neuer Eintrag hinzugefügt werden. Über das erscheinende Fenster können alle notwendigen Informationen zur Position eingetragen werden. Alle Felder bis auf den Einzelpreis sind Pflichtfelder und müssen ausgefüllt werden.
Der neue Eintrag wird mittels "Speichern" bestätigt und dem Wareneingang hinzugefügt.
Achtung: Das Hinzufügen eines Warenpostens ist nur möglich, wenn der zu bearbeitende Wareneingang noch nicht verbucht wurde.


Eine bestehende Position bearbeiten:

Mithilfe des Buttons "Bearbeiten" kann die ausgewählte Position bearbeitet werden. Es erscheint ein Fenster in dem die Änderungen getätigt werden können.
Achtung: Handelt es sich um einen verbuchten Wareneingang, kann nur mehr das Feld "Einzelpreis" geändert werden.


Anzahl der Verpackungseinheiten ändern:

Mithilfe des Buttons "VE-Vertielung Bearbeiten" kann für die ausgewählte Position die Anzahl der erhaltenen Verpackungseinheiten verändert werden.


Eine Position löschen:

Mithilfe des Buttons "Löschen" kann die ausgewählte Position entfernt werden.
Achtung: Das Löschen von Positionen ist nur möglich, wenn der Wareneingang noch nicht verbucht wurde.


Positionen sortieren:

Mithilfe der Pfeilbuttons kann der gewählte Eintrag innerhalb des Wareneingangs sortiert werden.


Wareneingang speichern:

Mithilfe des Buttons "Alles zum Depot hinzufügen" kann der bearbeitete Wareneingang im System gespeichert werden. Alle Eingaben werden erst hierdurch gültig.

VE-Verteilung bearbeiten

Diese Maske ermöglicht das nachträgliche Verändern der VE-Anzahl für einen eingehenden Warenposten.
Sie kann über den Button "VE-Verteilung Bearbeiten" beim Bearbeiten eines bestehenden Wareneingangs aufgerufen werden.


Verteilung bearbeiten:

Auf der linken Seite wird die eingehende VE-Anzahl samt Lieferant angezeigt.
Auf der rechten Seite wird angezeigt, wieviele Verpackungseinheiten aktuell den jeweiligen Kunden bzw. dem Depot zugeordnet sind.
Muss die Anzahl der Verpackungseinheiten nachträglich geändert werden, können die Zahlen dementsprechend angepasst werden.
Soll die neue Aufteilung bestätigt werden kann dies mittels Klick auf "Speichern" durchgeführt werden.


Artikel entfernen:

Handelt es sich um einen verbuchten Wareneingang und der gewählte Artikel muss vollständig aus dem System entfernt werden, kann dies durch den Button "Artikel und Verteilung entfernen" geschehen.
Hierdurch wird der Artikel aus dem Wareneingang und aus allen verbuchten Warenausgängen entfernt.
